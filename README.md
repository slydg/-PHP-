# 一个物物交换的平台 #

## 基本信息 ##
- 网站后端使用PHP编写，服务器使用XAMPP。
- 数据库使用MySQL（MariaDB）。数据库结构参见 SQL-structure.sql 文件。
- 曾使用网址：www.irongyi.cc 有过一定的使用记录，经测试相对稳定。

## 功能 ##
- 登陆注册；以及基本的用户信息管理，包括头像上传和裁剪等。
- 基本的物品信息管理，包含上传物品图片，描述等。
- 申请交换功能，包含了线上聊天，登陆后首页申请信息的实时推送等。此处使用了服务器推技术。
- 反馈渠道，物品分类展示等。
- 使用HTML5，经测试可以在多种平台、浏览器，以不同的分辨率正常显示。
- 个人信息使用“加盐”以及哈希表方法加密。

## 缺陷 ##
- 安全性能不足。
- 缺乏图片压缩功能，图片较大时，网站速度较慢。
- 第一次独立完成网站制作，难免会出现的或多或少的bug，以及网站设计的不足。
